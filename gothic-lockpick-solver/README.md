# Gothic Remake Plate Puzzle Solver

Статическое веб-приложение для игровой мини-игры с пластинами в Gothic Remake.

## Что делает

- выбирает количество пластин от 3 до 6;
- позволяет задать стартовое положение каждой пластины от 1 до 7;
- позволяет описать связи между пластинами через таблицу влияний;
- ищет последовательность ходов до центрального положения 4 для всех пластин;
- выводит готовую последовательность стрелок, чтобы ей можно было следовать в игре.

## Как запустить

Открой файл `gothic-lockpick-solver/index.html` в браузере.

Сборка не нужна. Приложение использует D3.js через CDN для отрисовки пластин.

## Как читать таблицу влияний

Строка означает пластину, которую игрок двигает вправо.

Столбец означает пластину, которая при этом меняется.

Значения ячеек:

- `—` нет связи;
- `→` пластина двигается в ту же сторону;
- `←` пластина двигается в противоположную сторону.

Например, если при движении пластины 1 вправо пластина 3 уходит влево, в строке `Пластина 1` и столбце `Пластина 3` нужно выбрать `←`.

## Алгоритм

Решатель использует поиск в ширину по состояниям. Ход считается допустимым, только если после него все пластины остаются в диапазоне 1–7. Первое найденное решение является самым коротким по количеству движений пластин при заданной таблице связей.
