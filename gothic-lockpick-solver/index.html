Gothic Remake helper

Решатель замков с пластинами

Введи стартовые положения и связи между пластинами. Приложение найдёт порядок ходов, чтобы вывести все пластины в центральный четвёртый слот.

1. Настрой замок

Пластин может быть от 3 до 6. У каждой 7 слотов, цель — слот 4.

2. Начальные положения

Нажми на слот каждой пластины. Жёлтый слот — текущее положение.

3. Связи между пластинами

Строка — какую пластину двигаем вправо. Столбец — какая пластина меняется. Ячейка переключается по клику: нет связи, вправо, влево.

4. Получить последовательность

Решатель перебирает допустимые состояния. Ход отбрасывается, если пластина выходит за границы 1–7.

Результат

    Как пользоваться

    1. Выбери количество пластин.
    2. Отметь стартовый слот каждой пластины.
    3. Заполни таблицу влияний для движения вправо.
    4. Нажми «Решить замок» и повтори последовательность стрелок в игре.

    Пример: если при движении пластины 1 вправо пластина 3 уходит влево, в строке «Пластина 1» и столбце «Пластина 3» поставь «←».
